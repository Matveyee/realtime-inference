Этот тестовый пример позволяет измерять скорость инференса моделей yolo на ускорителе RKNN, регулировать входную скорость кадров и следить за отставанием нейроести от реального времени

Сборка:
mkdir build
cd build
cmake ..
make

Пример использования

sudo ./main --model-path ../yolov6n.rknn --video-path ../Tennis1080p.h264 --frame-rate 20

Программа выводит задержку ( в кадрах ) между полученными кадрами и обработанным нейросетью кадрами
