Этот тестовый пример позволяет измерять скорость инференса моделей yolo на чипе hailo, регулировать входную скорость кадров и следить за отставанием нейроести от реального времени

Сборка:
mkdir build
cd build
cmake ..
make

Пример использования

sudo ./main --model-path ../yolov6n.hef --video-path ../Tennis1080p.h264 --frame-rate 40

Программа выводит задержку ( в кадрах ) между полученными кадрами и обработанным нейросетью кадрами
